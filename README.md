---

## 📖 About the Project

**Angkorism** is a dedicated tourism website built to provide visitors with a comprehensive and intuitive guide to exploring Siem Reap. From the iconic Angkor Wat complex to local experiences off the beaten path, the platform offers detailed destination listings, practical travel tips, and curated tour guides. 

The goal of this project is to create a seamless, visually engaging, and highly performant digital companion for anyone planning their journey to the heart of Cambodia's heritage.

## ✨ Features

* **Destination Directory:** Browse an extensive list of temples, museums, markets, and natural attractions in Siem Reap.
* **Curated Tour Guides:** Step-by-step itineraries and guides to help users maximize their visit.
* **Modern UI/UX:** A clean, minimalist interface optimized for seamless browsing.
* **Responsive Design:** Fully accessible and visually appealing on both desktop and mobile devices.

## 🛠️ Built With

* **Markup:** HTML5
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
